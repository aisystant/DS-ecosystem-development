## Назначение

Этот раздел описывает **программы развития созидателя** — структурированные контуры систематического саморазвития на трёх уровнях:
- Личное развитие (ЛР)
- Рабочее развитие (РР)
- Исследовательское развитие (ИР)

### Целевые метрики программы ЛР

| Метрика | Целевой минимум |
|---------|-----------------|
| Личная стратегия | План 1 мес–3 года |
| Экзокортекс | ≥300 заметок, ≥20 заготовок |
| Инвестированное время | ≥200 ч логов |
| Практики ученика | 8/8, удержание ≥70% |
| Системные модели | ≥2 модели |
| Телесная собранность | 32 ч тренинга, 3 теста |

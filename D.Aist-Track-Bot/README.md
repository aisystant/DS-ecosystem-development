# Ядро D: Aist Track Bot (Telegram-бот персонального маршрута)

## Обзор ядра

**SoI (Система интереса)**: Telegram-бот персонального маршрута обучения — AI-ассистент для развития системного мышления.

**Репозиторий бота**: [github.com/aisystant/aist_track_bot](https://github.com/aisystant/aist_track_bot)

## Системная структура ядра D

```
D.Aist-Track-Bot/
├── D1.IT-Platform/           # Надсистема (ИТ-платформа содержит бота)
│   ├── D1.1.Meaning/         # Зачем платформе нужен бот
│   ├── D1.2.Architecture/    # Как бот интегрируется в платформу
│   └── D1.3.Operations/      # SLA, требования к боту

│   ├── D2.1.Meaning/         # Миссия, ценностное предложение
```

## Граф системных отношений

```
ИТ-платформа Экосистемы (D1) ∋ Aist-Track-Bot (D2)
                                      ↑
                                      │ создаёт
                                      │
                              ИТ-служба (D3)
```

**Связь с другими ядрами:**
- D1 (ИТ-платформа) = C2 (детализация подсистемы ИТ-платформы)
- D3 (ИТ-служба) = C3 (та же система создания)
- Бот служит созидателю (A2) через экосистему (B2)

## Текущее состояние

Бот функционирует в двух режимах:
1. **Марафон** — 14-дневная интенсивная программа
2. **Лента** — гибкий формат по выбранным темам

Планируется третий режим:
3. **Экзокортекс** — режим сборки системы создания мышления

## Технологический стек

- Python 3.11 + aiogram 3.x
- Anthropic Claude API
- PostgreSQL
- Railway / Render
